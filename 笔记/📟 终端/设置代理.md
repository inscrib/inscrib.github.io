## 设置代理

限于 Linux 与开启了代理的机器在同一个**局域网**或者 Linux 作为**子系统**的情况。

```shell
export ALL_PROXY="http://<地址>:<端口>"
```

例如：

```shell
export ALL_PROXY="http://127.0.0.1:7890"
```

> 可以在 Clash 中使用复制对应的**端口和地址**，然后开启 **Allow LAN** 允许局域网链接
